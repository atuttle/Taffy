The REST Web Service Framework for ColdFusion and Railo

Download .zip Download .tar.gz View on GitHub

You're here because creating REST APIs with the native functionality in ColdFusion 10 and Railo is verbose, complex, and developer-hostile. Or maybe you're still on an older version of ColdFusion. You've come to the right place.

Taffy is low friction, extremely simple to get started, and it's compatible as far back as ColdFusion 8.

Taffy is terse because it uses convention over configuration, and doesn't require writing a bunch of boilerplate code. How terse? A functional API can fit into a tweet:

component extends="taffy.core.api"{} component extends="taffy.core.resource" taffy_uri="/hi"{function get(){return representationOf('hi');}}

— Taffy (@cf_taffy) April 25, 2013

It's easy to debug and it optionally integrates with your favorite IoC libraries like ColdSpring and DI/1.
